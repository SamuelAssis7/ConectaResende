# Conecta Resende

Plataforma web que conecta moradores da cidade de Resende (RJ) a oportunidades de:

- Emprego
- Cursos
- Ações sociais

## Funcionalidades

✔ Cadastro e login de usuários  
✔ Criação de publicações  
✔ Filtro por tipo (Emprego, Curso, Social)  
✔ Contato por email com o autor da vaga  
✔ Área de perfil do usuário  

## Tecnologias utilizadas

- ASP.NET Core
- C#
- PostgreSql
- Bootstrap
- JavaScript

## Objetivo

Criar uma plataforma simples que facilite a divulgação de oportunidades na cidade de Resende.
